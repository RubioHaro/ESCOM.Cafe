Registro

Realizar pedido

Para realizar pedido, necesitas recargar dinero a nuestro sitio. Esto cuenta como dinero electrico para compra con nosotros. Para esto, debes anexar un archivo de formato jpg o pdf de screenshot de comprobante de déposito bancario. Cuando mosotros lo verificamos, te actualizamos saldo. Este proceso se realiza dentro de 24 horas máximo.
